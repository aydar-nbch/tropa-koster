# Интернет-магазин туристического снаряжения "Тропа — Костер"

Этот проект представляет собой полнофункциональный интернет-магазин товаров для туризма, кемпинга и активного отдыха. Проект был разработан в рамках прохождения производственной практики.

## Основные возможности

Каталог товаров: Удобная фильтрация по категориям, поиск по названию и сортировка по цене/рейтингу.

Корзина: Динамическое управление составом заказа, пересчет стоимости и интеграция с локальным хранилищем (localStorage).

Авторизация: Безопасная регистрация и вход пользователей с использованием JWT-токенов.

Личный кабинет: История заказов, редактирование профиля пользователя.

Администрирование: Панель управления для добавления, редактирования и удаления товаров.

Блог: Раздел статей для туристов с возможностью управления контентом.

Оформление заказа: Интеграция формы заказа и логика взаимодействия с базой данных.

🛠 Технологический стек (MERN Stack)

Frontend: React.js (Vite), Redux Toolkit (состояние), Axios (запросы), Lucide React (иконки).

Backend: Node.js, Express.js.

База данных: MongoDB, Mongoose (ORM).

Безопасность: JWT (JSON Web Tokens), bcryptjs (шифрование паролей).

## Установка и запуск

Предварительные требования

Убедитесь, что у вас установлены Node.js и MongoDB.

Шаги по запуску

Клонируйте репозиторий:

git clone https://github.com/aydar-nbch/tropa-koster.git cd название-проекта

Настройка сервера (Backend):

cd backend npm install

Создайте файл .env в папке server и добавьте переменные:

PORT=5000 MONGO_URI=ваша_строка_подключения_к_mongodb JWT_SECRET=ваш_секретный_ключ

Запустите сервер: npm run dev

Настройка клиента (Frontend): В новом терминале перейдите в папку клиента:

cd ../frontend npm install npm run dev

Приложение будет доступно по адресу http://localhost:5173.

## Структура проекта

/frontend — React-приложение.

/backend — Express-сервер, контроллеры и модели БД.

/db_dump - данные с базы данных в формате *.json.

/uploads — папка для хранения изображений товаров (если используется локальное хранилище).

## Развертывание базы данных (MongoDB)

Для полноценной работы проекта вам необходимо импортировать данные базы данных. Для этого выполните следующие действия в **MongoDB Compass**:

1. Откройте **MongoDB Compass** и подключитесь к вашему локальному серверу MongoDB.
   
2. Создайте новую базу данных с именем `tropa-koster`.
   
3. Внутри базы данных создайте следующие коллекции:
   * `orders`
   * `posts`
   * `products`
   * `reviews`
   * `users`
     
4. Для каждой коллекции выполните импорт данных:
   * Нажмите на название коллекции -> кнопка **Import Data**.
   * Выберите соответствующий файл из папки `/database` в этом репозитории:
     * Для `orders` — `tropa-koster.orders.json`
     * Для `posts` — `tropa-koster.posts.json`
     * Для `products` — `tropa-koster.products.json`
     * Для `reviews` — `tropa-koster.reviews.json`
     * Для `users` — `tropa-koster.users.json`
       
5. После выбора файла нажмите **Import**, чтобы загрузить данные.
